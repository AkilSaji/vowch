# Vowch client portal

Responsive browser portal mirroring the Vowch mobile experience: discover and post
gigs, apply, manage jobs, view chat/activity, wallet, trust passport, Houses, and
safety status. It runs with the same seeded demo data as the mobile app.

```powershell
cd apps/client-portal
npm install
npm run dev
```

For Vercel, create a project with Root Directory `apps/client-portal` and Framework
Preset `Vite`. Its free `*.vercel.app` URL is suitable for hackathon judges; a custom
domain is not required.
